# VGG Face Descriptor
VGG-Face Descriptor port to pytorch.

1. Download original luatorch weights from http://www.robots.ox.ac.uk/~vgg/software/vgg_face/
2. Execute demo: ```python3 models/vgg_face.py```. It should print a 2.

Enjoy.
